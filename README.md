# Bloss1

Final project for my Computer Science course - UnB

C++ game engine.

:)

> Note: the build/run commands use the [just command runner](https://github.com/casey/just)

## Build & Run

```
$ just
```

## Build

```
$ just build cfg
```

## Run

```
$ just run cfg
```

## Clean

```
$ just clean cfg
```
